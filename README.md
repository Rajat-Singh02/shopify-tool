# Shoppable Video

Manual-upload shoppable video app for Shopify merchants.

Feature 0 provides the project foundation, shared packages, Prisma schema, Polaris admin shell, and baseline tests. Later features will add Shopify authentication, product search, upload, processing, widgets, cart actions, and analytics.
